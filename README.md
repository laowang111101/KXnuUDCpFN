# 员工管理系统简介 java402

## 项目概述

员工管理系统是基于Springboot和Vue的前后端分离系统，旨在提供用户和管理员之间的便捷交互。本项目适用于学习和企业内部管理，帮助提高工作效率。

## 技术栈

- 后端：Java, Springboot, MyBatis, Maven
- 数据库：MySQL 5.7
- 前端：Vue.js 12版本

## 功能模块

系统主要分为以下两个角色：

- **用户**
  - 用户登录
  -个人信息管理

- **管理员**
  - 员工信息管理
  - 用户管理

## 系统角色

- **用户**：普通员工，可以查看和编辑个人信息。
- **管理员**：负责管理所有员工的信息，以及用户账号的维护。

## 运行环境

- JDK 1.8
- Maven
- MySQL 5.7

## 部署步骤

1. 准备开发环境：JDK 1.8, Maven, MySQL 5.7。
2. 克隆或下载源代码。
3. 导入数据库文件。
4. 编译并运行项目。
